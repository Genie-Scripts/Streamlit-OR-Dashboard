🏆 ハイスコア計算方法（100点満点）

診療科ランキングの総合スコアは、以下の3つの指標から構成されています：

1. 🎯 全身麻酔手術件数（70点満点）- 最重要指標

週単位の全身麻酔手術件数（麻酔時間20分以上）を多角的に評価します。

配点内訳：
  • 直近週達成度（30点）
    • CSV目標値に対する達成率で評価
    • 達成率100%以上：30点
    • 達成率90-99%：24点
    • 達成率80-89%：18点
    • 達成率70-79%：12点
    • 達成率70%未満：0-6点
  • 改善度（20点）
    • 評価期間の平均と過去期間の平均を比較
    • 改善率+20%以上：20点
    • 改善率+10-19%：15点
    • 改善率+5-9%：10点
    • 改善率0-4%：5点
    • マイナス成長：0点
  • 安定性（15点）
    • 週次実績の変動係数で評価
    • 変動係数10%未満：15点（非常に安定）
    • 変動係数10-20%：12点（安定）
    • 変動係数20-30%：8点（やや不安定）
    • 変動係数30-40%：4点（不安定）
    • 変動係数40%以上：0点（極めて不安定）
  • 持続性（5点）
    • 週次トレンドの傾きで評価
    • 上昇トレンド：5点
    • 横ばいトレンド：3点
    • 下降トレンド：0点

2. 📊 全手術件数（15点満点）

診療科の全体的な手術活動量を評価します。

配点内訳：
  • 診療科間ランキング（10点）
    • 1位：10点
    • 2位：8点
    • 3位：6点
    • 4位：4点
    • 5位：2点
    • 6位以下：0点
  • 改善度（5点）
    • 前期比+10%以上：5点
    • 前期比+5-9%：3点
    • 前期比0-4%：1点
    • 前期比マイナス：0点

3. ⏱️ 総手術時間（15点満点）

手術室の稼働効率と貢献度を評価します。

配点内訳：
  • 診療科間ランキング（10点）
    • 1位：10点
    • 2位：8点
    • 3位：6点
    • 4位：4点
    • 5位：2点
    • 6位以下：0点
  • 改善度（5点）
    • 前期比+10%以上：5点
    • 前期比+5-9%：3点
    • 前期比0-4%：1点
    • 前期比マイナス：0点

⚠️ 手術時間の計算方法：

  • 入室時刻から退室時刻までの経過時間
  • 深夜跨ぎ対応（23:30入室→1:15退室 = 1時間45分）

📊 総合スコア = 全身麻酔(70点) + 全手術(15点) + 手術時間(15点)

※ 最高100点満点で評価

グレード判定：
  • S 90点以上（卓越したパフォーマンス）
  • A 80-89点（優秀なパフォーマンス）
  • B 70-79点（良好なパフォーマンス）
  • C 60-69点（標準的なパフォーマンス）
  • D 60点未満（改善が必要）
